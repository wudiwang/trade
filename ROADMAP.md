## 0. 当前策略（chan_bi，2026-06-13 起，取代V4定义）
- **一买**：前期下跌**成一笔**(≥5根合并K,去包含) → 末端**底分型 + 停顿K**(A路)，**或** 放量≥3x下跌K被收回(B路,仍需前期成笔)。
- **二买**：一买后 上涨成笔 + 下跌成笔 → 第二笔末端**底分型(更高低点) + 停顿K**。做空镜像。双向，5m/15m/1h。
- 入场=停顿K收盘(A)/放量K顶部(B)；止损=分型/放量K极值外；止盈=下跌前顶/密集区。
- 假设(待用户验证)：停顿K=收盘>底分型右K最高价；二买=更高的低点。strategy=chan_bi(可切回spring_v4/chan_v1)。
- paper 模拟盘 + 回测 + Web控制台 + TG提醒 + 预演式K线标注。部署于 VPS(76.13.182.175)。

## 1. 用户的核心理念（纲领，所有策略的指导）
- **大盘第一**：下跌趋势中任何上涨都可能酝酿更强下跌（反弹=诱多）。做任何信号都要先看BTC大盘方向。
- **流动性买点**：最好的买点 = 价格"破掉"关键位(前高/前低/流动性所在) → **假突破再跌回收**(sweep + reclaim)。这正是V4策略的灵魂，要继续强化为最高优先级买点。
- **大势研判源**：YouTube博主「提阿非罗」每日(周一~周五)更新BTC大盘视频，从日线/周线/4h做方向研判，指出流动性所在和好的介入位。其观点作为重要参考/提醒点。
- **顺藤摸瓜**：BTC研判 → 顺势研究 ETH/SOL 大市值币 → 再到优质山寨。
- **操纵识别**：山寨币持仓数据(大户私有比例、筹码集中)是判断"是否被操纵/能否逼空暴涨"的关键。如 PEPE：链上显示大部分代币被大户私有、流通极少 → 合约价差大、费率贵 → 暴涨后继续暴涨。
- **多维消息面**：宏观消息 + 社群讨论热度 + KOL带货 作为关注/加分因子（讨论声高 + K线低位 → 进关注列表）。
